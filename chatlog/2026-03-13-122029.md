## 三、以后连接 Mac mini 的简洁方法

以后最简单的方式，直接在 **MacBook** 上运行：

```bash
OPENCLAW_DASHBOARD_TOKEN='6c07ffdb5497a6975efd0794267d0eeae23be2fc1f80fa9f' \
OPENCLAW_REMOTE_HOST='linqingxuandeMac-mini.local' \
OPENCLAW_REMOTE_USER='linqingxuan' \
zsh ~/.openclaw/shared/runtime/remote_ops/macbook_dashboard_tunnel.sh --open
```

效果是：

1. 在后台建立 SSH tunnel
2. 把 MacBook 本机 `127.0.0.1:18790` 映射到 Mac mini 的 `127.0.0.1:18789`
3. 自动打开浏览器

以后浏览器里访问的就是：

- `http://127.0.0.1:18790/#token=6c07ffdb5497a6975efd0794267d0eeae23be2fc1f80fa9f`

### 最重要的使用纪律

这条命令必须在 **MacBook** 上运行，不要在 Mac mini 上运行。

如果只想手动开 tunnel，不自动打开浏览器：

```bash
zsh ~/.openclaw/shared/runtime/remote_ops/macbook_dashboard_tunnel.sh
```

如果想让它在后台挂着，不占当前终端：

```bash
zsh ~/.openclaw/shared/runtime/remote_ops/macbook_dashboard_tunnel.sh --background
```
